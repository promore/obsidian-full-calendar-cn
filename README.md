# obsidian-full-calendar-CN

该版本是对官方插件的中文汉化版本。
使用 trae AI code 自动编码的，本人已经在自己的机器上测试过，请放心服用！

-   如果你对源码没有兴趣或者想直接使用该汉化插件，请下载 main.js & styles.css 替换掉插件目录下的文件， manifest.json 文件可选。
-   保险起见，你最好先备份一下你的插件目录下的文件。
-   备份完成后，将 main.js & styles.css 替换掉插件目录下的文件。
-   重启 Obsidian。
